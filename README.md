# Демо https://uxdimon.github.io/vasyan/map.html

## Использование

Устанавливаем

```
npm install
```

Сборка для разработки

```
gulp
```

Сборка для продакшена

```
gulp prod
```

Обновить библиотеки

```
gulp up-lb
```
